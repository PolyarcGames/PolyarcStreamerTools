# Polyarc Streamer Tools

A collection of tools and configurations to help streamers and content creators showcase Polyarc games with enhanced camera controls and Stream Deck integration.

## What's Included

### Glassbreakers Web Panel
A web-based control panel for managing spectator camera settings in Glassbreakers. Features include:
- **Spectator Camera Controls**: Toggle spectator camera mode and cycle through different camera mounts
- **Auto-launch Options**: Automatically show the control panel when needed
- **Remote Control**: JavaScript-based API for controlling camera mounts and spectator settings

See full documentation [here](https://github.com/PolyarcGames/PolyarcStreamerTools/wiki/Glassbreakers-(EN))

### Elgato Stream Deck Profile
Pre-configured Stream Deck profile (`Glassbreakers.streamDeckProfile`) for quick access to Glassbreakers streaming controls.

Simply import the Stream Deck profile from the [ElGatoStreamDeck](ElGatoStreamDeck) folder into your Elgato Stream Deck software

See full documentation [here](https://github.com/PolyarcGames/PolyarcStreamerTools/wiki/ElGato-Stream-Deck)
